**Flask App**
Download all the below given things :
1. flask
2. flask-sqlalchemy
3. flask-session
4. psycopg2
5. requests

**Steps to start the app**
1. Make a venv : python -m venv venv
2. activate the venv : venv/Scripts/activate
3. Install all the packages : pip install package_name
4. run this then : python run.py
